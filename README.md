# Budgetly

Budgetly is a personal finance web app built with Next.js that helps users track spending, manage wallets, monitor debts and savings goals, and get AI-powered financial guidance.

## What this app does

- Track income and expense transactions
- Organize money across multiple wallets
- Transfer funds between wallets
- Manage debts and record repayments
- Set savings goals and track contributions or withdrawals
- Create recurring transaction templates
- View spending insights and financial health indicators
- Chat with an AI assistant for personalized finance guidance

## Tech stack

- Next.js 16 (App Router)
- React 19 + TypeScript
- Firebase Authentication + Firestore
- Tailwind CSS v4
- Radix UI + custom UI components
- Google Gemini API for chat assistant
